# Shared Structs C#/HLSL (Unity)
Automatically generates .hlsl equivalent struct files from your C# structs. By simply adding the [SharedStruct] attribute to any C# struct, equivalent .hlsl files will be automatically generated with no extra work. Structs are automatically re-ordered in order of dependency, and automatically #includes any referenced external struct also marked with the [SharedStruct] attribute. 

Generated files are written to 'Assets/Auto Generated HLSL Structs'. For arrays, either add the fixed keyword and specify a length, or [MarshalAs] the type with a constant length value. Only 32-bit built-in types are valid. Struct functions/constructors are not converted, only variables.

## Installation

Simply add the entire folder into your project's Assets folder.

## Example Inputs & Automatically Generated .HLSL Files

## FileA.cs:
```csharp
[SharedStruct]
public struct MyStructA
{
    public int a;
    public uint b;
    public Vector3 c;
    public Vector2Int d;
    public MyStructB e;
}

[SharedStruct]
public struct MyStructB
{
    public Vector3 a;
    public Vector3 b;
    public Vector3 c;
    public MyStructC d;
}
```
## FileB.cs:
```csharp
[SharedStruct]
public struct MyStructC
{
	int a;
	int b;
	MyStructD c;
}

[SharedStruct]
public struct MyStructD
{
	int a;
	int b;
}
```
## FileA_AutoGenerated.hlsl:
```hlsl
#ifndef MyFileA_AutoGenerated
#define MyFileA_AutoGenerated

#include "MyFileB_AutoGenerated.hlsl"

struct MyStructB {
	float3 a;
	float3 b;
	float3 c;
	MyStructC d;
};

struct MyStructA {
	int a;
	uint b;
	float3 c;
	int2 d;
	MyStructB e;
};

#endif
```
## FileB_AutoGenerated.hlsl:
```hlsl
#ifndef MyFileB_AutoGenerated
#define MyFileB_AutoGenerated

struct MyStructD {
	int a;
	int b;
};

struct MyStructC {
	int a;
	int b;
	MyStructD c;
};

#endif
```
